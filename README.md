# sepatu

Perancangan Inventori Sepatu (Progress)

- Hak Akses (Admin)
- Aplikasi dapat melakukan CRUD (Create, Read, Update, Delete)
- Admin dapat Menambahkan Tranasaksi (Stock In)

**Teknologi**

- bootstrap 4
- adminlte
- sweetalert

- php native
- javascript
- ajax
- jquery

Database
SQL : phpmyadamin
nama : db_sepatu.sql
location : sepatu/database/db_sepatu.sql

username: admin 
password: admin
